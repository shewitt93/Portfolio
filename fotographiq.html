Fotographiq, photo blogging service. This site was designed as a way for people to post pictures they've taken anonymously, allowing other users ineractivity with the photo by commenting and emoting it. Our site allows users to upload any size image files with descriptions and a title. Using Bulma to create the modal you see to the right, the user has a selection of options regarding type of camera, lens and format. They can also insert Gifs instead of a photo. The file selection is local only currently and should accept any image format. The image is stored using base64 encoding. After a post has been made users can comment, like, and react to the photo. There is also the functionality to delete and edit the post after it has been made. You can also search by tag to narrow down your selection to format, camera type and lens used!
